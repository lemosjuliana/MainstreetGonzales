# Main Street Committee Website

## Project Overview

The Main Street Committee Website will serve residents who want to:
- View upcoming Main Street events
- Discover local businesses
- Contact the Main Street Committee 
- View information and access forms to apply for the Gonzales Main Street Advisory Board
- View information and access forms for the Gonzales Main Street Incentive Program
- View information and access forms for the Honor Our Veterans Banner Project
